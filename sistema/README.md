Bienvenidos al sistema de rentas de peliculas

Para correr este proyecto es necesario contar con:
    - Apache
    - PHP
    - MySQL
    - Git
    - Remondado: XAMPP

Pasos a seguir para correr el proyecto:
    - git clone 
    - composer install
    - npm install
    - php artisan migrate
    - php artisan db:seed
    - php artisan serve
